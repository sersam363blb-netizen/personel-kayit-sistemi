# Personel Kayıt Sistemi

C# Windows Forms kullanılarak geliştirilmiş, veritabanı tabanlı bir personel yönetim uygulamasıdır.

## Proje Yapısı
- 1 Veritabanı
- 2 Tablo
  - Yönetici
  - Çalışan (Personel)

## Sayfalar
- Giriş Sayfası (Yönetici girişi)
- Personel İşlem Sayfası
  - Listeleme
  - Kayıt ekleme
  - Silme
  - Güncelleme
- İstatistikler Sayfası
- Grafikler Sayfası

## Özellikler
- Personel bilgileri CRUD işlemleri
- Medeni durum, maaş, şehir ve meslek bilgileri
- DataGridView ile listeleme
- Grafik ve istatistik ekranları
- Veritabanı bağlantılı yapı

## Kullanılan Teknolojiler
- C#
- Windows Forms
- SQL Server
